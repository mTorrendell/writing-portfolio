# writing-portfolio

Single-page writing portfolio for Mercedes Torrendell — writer and interviewer on AI, language, and people. Published at The Creative Independent.

- One file: `index.html` (embedded CSS, near-zero JS, Google Fonts only).
- Warm monochrome, hairlines only, sentence case. Design brief in `CLAUDE.md`.
- Deploys as a static site on Vercel. Custom domain (later): `writing.thesphereimpact.com` via CNAME in Squarespace DNS.
